# Eon-bot
A modular Discord bot written with the [JDA](https://github.com/DV8FromTheWorld/JDA) library.

## Compiling
Install JDK 8.

### Windows
Compile: `gradlew uberjar`\
Run: `gradlew run`

### *nix
Compile: `./gradlew uberjar`\
Run: `./gradlew run`
